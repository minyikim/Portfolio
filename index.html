SKILL

HTML5

웹 표준·웹 접근성을 준수하며, HTML
마크업 방식과 시맨틱 태그를 사용하여
전체적인 레이아웃을 작성 가능합니다.

CSS3

IE 지원 까지 고려하여 Cross Browsving을 위하여 기존의 ‘float’와 최신의 ‘flex’, ‘Grid’ 기법을 활용 또한, 미디어쿼리를 이용하여 반응형 웹 구현 가능합니다.

Js & jQuery

필수적인 자바스크립트 활용이 가능하며 내비게이션, 탭 메뉴, 모달창 등을 순수 jQuery 코딩으로만 제작 가능합니다.

Illustrator

로고디자인 및 그래픽 디자인, 아이콘 디자인,
명함 디자인, 와이어프레임 시안 등의
기본 일러스트 구현 가능합니다.

Photoshop

이미지 편집 및 합성, 보정 등 구현
웹 퍼블리셔 직군 특성에 맞게
기본 툴 사용 가능합니다.

Lightroom

풍경 이나 사물 이미지 편집 및 보정,
관리 등의 편집 가능합니다.

C Programming

C 프로그래밍 언어의 기본문법에 숙지하고 있습니다. 이외 중등학교 (정보컴퓨터) 2급 정교사를 소지하며, 프로그래밍에 대한 교육기획 및 공학을 가능합니다.

VSCODE

주로 Visual Studio Code을 사용하며 이외에도 Notepad++, Visual Studio도 사용 가능합니다.

GitHub

퍼블리싱 한 내용을 공개하며,
Git 호스팅 프로그램에 활용 가능합니다.
